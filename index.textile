Welcome to the official gitg homepage.

To understand what gitg does, you should know what version control is and what git is.

Other git clients include: gitk, gitx, tortoise. And then there is the trusted command line interface.

  
h1. gitg Features

gitg offers the following features:
* visually browse a git commit log
* do commits (with graphical selection of files to commit)
